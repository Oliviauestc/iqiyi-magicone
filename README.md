# magic-one-iqiyi

爱奇艺全国高校技术大赛web作品
利用主办发爱奇艺提供的API，开发了一个轻社交的视频网站，包括主页、搜索、排行榜、等功能。


## 一、安装说明

 - 本项目运行在node v6.11.1下。直接git clone，然后进入项目目录，运行node server.js即可。

##二、技术栈

 - express4.15.3 提供路由处理
 
 - vue2.0提供数据渲染

 - bootstrap提供响应式布局
